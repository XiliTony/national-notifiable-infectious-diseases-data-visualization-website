# 全国法定传染病疫情数据可视化网站

## 项目简介

公共卫生安全是国家安全的重要组成部分，对法定传染病的有效监控、分析和预警是保障人民健康的关键环节。相关数据以其时效性、地域性和规律性，成为洞察疾病传播趋势、评估防控政策效果、指导公众科学防护的宝贵信息源。

本项目旨在利用Web前端可视化技术，构建一个交互式的“**全国法定传染病数据分析系统**”。

---

## 技术架构

本项目采用 **前端Web开发技术**：

### 前端
| 技术 | 用途 |
|------|------|
| HTML5 | 构建用户界面 |
| CSS3 | 渲染页面样式 |
| JavaScript | 实现数据交互 |
| ECharts | 完成可视化图表 |

---

## 项目亮点
- **数据可视化**：页面插入多种图表，对传染病疫情数据进行了全面直观的展示。
- **二级页面**：把光标放在头部右侧的灰色导航栏中，会根据内容出现不同的柱状图并把散点图盖住，这就是二级页面，能使用户很方便的查看不同传染病的数据。
- **按年度切换折线图**：在折线图区域，点击右上角的不同年份，会切换不同的数据，在有限的空间里放入了更多的数据。
